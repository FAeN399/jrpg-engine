## Intended Purpose

This directory separates **game-specific** content from **reusable engine/framework** code:

### Engine vs Game Separation
```
engine/       <- Reusable core (any game type)
framework/    <- JRPG-specific reusable systems
game/         <- Specific game using the above
```
